## 🔧 技术实现

### 状态管理
使用 Svelte 5 的 `$state()` 管理所有响应式状态：
- `currentPage`：当前页码
- `scale`：缩放比例
- `loading`：加载状态
- `loadProgress`：加载进度
- `isFullscreen`：全屏状态
- `searchText`：搜索文本
- `searchResults`：搜索结果

### 事件处理
- 键盘事件：`handleKeydown()`
- 触摸事件：`handleTouchStart/Move/End()`
- 搜索事件：`handleSearch()`

### 性能优化
- 懒加载 PDF.js
- 动态导入减少初始包大小
- 进度回调优化大文件加载体验

## 🔍 故障排查

### 问题：PDF 无法加载
**可能原因**：
1. 文件路径错误
2. PDF 文件损坏
3. 跨域问题

**解决方案**：
1. 检查文件是否存在于 `public/pdfs/`
2. 尝试在浏览器中直接访问 PDF URL
3. 检查浏览器控制台的错误信息

### 问题：搜索功能不工作
**可能原因**：
1. PDF 是扫描件（图片）
2. PDF 没有文本层

**解决方案**：
- 搜索功能仅适用于有文本层的 PDF
- 扫描件需要 OCR 处理后才能搜索

### 问题：触摸手势不灵敏
**可能原因**：
1. 浏览器兼容性问题
2. 触摸事件被阻止

**解决方案**：
- 确保在移动设备上测试
- 检查是否有其他 JavaScript 阻止了触摸事件
